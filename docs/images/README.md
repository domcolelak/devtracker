# Screenshots

Screenshots embedded in the main README. To regenerate them, boot the stack
(see the Quickstart section of the main README), then:

- `django-admin.png` - open http://localhost/admin/ and log in as `admin` with
  the seed password, then capture the Teams changelist.
- `swagger-ui.png` - open http://localhost/api/docs, expand the POST /tasks
  operation and capture the page.
